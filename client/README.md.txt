## Description

Teste para FullStack developer pleno.

## Instalando dependências

A instalação das dependências é simples, podemos executar o script:

```bash
$ yarn
```

## Iniciando a aplicação

```bash
# Ambiente de desenvolvimento
$ yarn dev
```

## Funções atuais

- Até o momento foram criados a rota root com uma tela de login, onde é possível inserir informações de username e password para se fazer a autenticação, recebendo de volta um access_token com expiração de 1min que será armazenado no localStorage durante esse tempo.
- Projeto foi concluído até o início da segunda etapa onde usuário seria redirecionado para a rota de pagáveis, mas infelizmente não foi concluído.
